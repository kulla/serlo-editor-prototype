# Project Overview

A web-based WYSIWYG editor prototype built with React, ProseMirror, and Loro CRDT for collaborative editing. The project uses a schema-driven architecture with flat node storage for efficient CRDT operations.

* **Testing:** Currently no test suite is configured. This repo represents a technical prototype and therefore no tests should be added.
* **Checks:** Run `./node_modules/.bin/biome check` and `./node_modules/.bin/tsc --noEmit` after each code change.
